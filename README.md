👁️ MediAgent-Retina - Autonomous Retinal Diagnosis

An intelligent, deep learning-powered medical assistant designed to provide autonomous retinal disease detection. By combining state-of-the-art computer vision with Agentic AI reasoning, this platform transforms raw retinal scans into interpretable diagnostic insights with high precision.

✨ Key Features
Autonomous Agentic Reasoning: Beyond simple classification, the system uses Agentic AI to analyze findings, cross-reference medical patterns, and provide logical diagnostic explanations.

Deep Learning Pipeline: High-accuracy neural networks trained to detect anomalies such as Diabetic Retinopathy, Glaucoma, and Macular Degeneration from fundus images.

Interactive Diagnostic Interface: A modern, responsive dashboard for uploading scans and viewing real-time AI-driven analysis.

Automated Medical Reporting: Generates comprehensive, human-readable reports summarizing the severity and specific markers identified by the model.

Literature Integration: The agentic layer can simulate clinical workflows by contextualizing model outputs with established medical knowledge bases.

Secure Data Handling: Built with a focus on privacy, ensuring medical images are processed efficiently and securely within the diagnosis pipeline.

🛠️ Built With
Python - Core language for model training and agent orchestration.

Deep Learning (TensorFlow/PyTorch) - Powering the retinal image segmentation and classification.

Agentic AI Frameworks - Enabling autonomous reasoning and multi-step diagnostic workflows.

OpenCV - Advanced image preprocessing and retinal feature enhancement.

Streamlit / Flask - For building a seamless and interactive user interface.

LaTeX - Used for generating professional, formatted clinical reports.

🚀 Getting Started
To get a local copy of the diagnostic tool up and running, follow these steps:

Clone the repository:
git clone https://github.com/Anshuman14Pattanaik/MediAgent-Retina.git

Navigate to the project directory:
cd MediAgent-Retina

Install dependencies:
pip install -r requirements.txt

Run the application:
python app.py
(Or streamlit run app.py if using the Streamlit interface)
